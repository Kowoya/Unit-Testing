# Dominion of Shadows

Консольна гра, створена мовою C++ із використанням принципів об’єктно-орієнтованого програмування (ООП).

## Мета роботи
Метою є отримання практичних навичок:
- роботи з системою контролю версій Git та сервісом GitHub;
- написання юніт-тестів;
- застосування принципів інкапсуляції, успадкування та поліморфізму;
- формування повного робочого циклу розробки — від створення репозиторію до Pull Request.

## Основні можливості
- Керування: введення з клавіатури  
- Об’єкти гри: гравець, вороги, заклинання, здоров’я, досвід  
- ООП: класи `Character`, `Player`, `Enemy`; успадкування, поліморфізм, інкапсуляція  
- Тестування: юніт-тести для перевірки основних методів  

##  Тестування
Для тестування використовується фреймворк Catch2.
